How to enable Firebase Phone Authentication(OTP)

How to enable Firebase Phone Authentication(OTP)

  1. Open your firebase console.
  2. Go to Authentication and open Sign-in method.
  3. Enable Phone Sign-in method and save.

  1. Now, we have to add sha1 and sha256 key in firebase, for that perform below given step for that you can do one of the below method
  2. Go to the android folder. Right click gradlew file and select Open in Terminal as shown in below image.

  1. Go to the terminal view and paste- gradlew signingReport as shown in below
  2. For mac it should be ./gradlew signingReport

  1. Now take debug sha and sha256 will print copy that and add it in firebase console. And also add sha-256 to firebase console.

Remember:

  1. When we are releasing apk then we need to add release sha in firebase panel otherwise otp will not work.
  2. After release you can get sha from play console here

  1. Or another way you can also get release sha from below given command in cmd
  2. Go to run and type cmd
  3. Here first go to java jdk bin path
  4. Then here edit this command and replace your keystore path and alias in this command keytool -list -v -keystore "D:\keystore\eShop.jks" -alias eShop
  5. Then it will ask for password. Enter and then it will show your release sha
